API Documentation

Endpoint

GET /api/vehicle-count

Description

This API endpoint performs object detection on an image and returns the count of different vehicle types in JSON format.

Request

The request should be a JSON object with the following properties: 


{
  "image": "base64-encoded-image"
}

Note: The "image" property should contain the base64-encoded image data.

Response

The API response will be a JSON object with the following format: 


{
  "data": {
    "bicycle": 2,
    "car": 5,
    "motorcycle": 1,
    "bus": 0,
    "truck": 3
  }
}

The "data" property contains the counts of different vehicle types identified in the provided image.
